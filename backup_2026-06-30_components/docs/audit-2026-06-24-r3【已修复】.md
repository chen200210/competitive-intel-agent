## 🔴 会导致功能出错

### ✅ BUG #A1: 批量 AI 打分 ID 计算错误，第二批起全部归零

**文件**: `scripts/score_news.py:241` | **严重度**: 高 | **状态**: 已修复

```python
# 错误: i - start 对 batch 2+ 产生负值
for i, item in enumerate(batch, start=start):
    compact["id"] = i - start  # batch 2: start=15 → IDs: -15..-1
```

LLM 返回的 keys 是 `"0"`-`"14"`，但代码用 `str(i)` 去查 `raw_candidates`，其中 `i` 是 `enumerate(batch)` 从 0 开始的索引。批次 1 碰巧正确（start=0，i-start = i），批次 2 起全部不匹配 → 25 条新闻静默获得 `ai_score=0` 和空摘要。

**修复**: `i - start` → `i`（`enumerate` 的原生索引）。briefer.py 不拆批，无需改。

### ✅ BUG #A2: `dict.get(key, 0)` 对 null 值返回 None 导致 TypeError

**文件**: `scripts/score_news.py:278`、`src/agents/briefer.py:1575` | **严重度**: 高 | **状态**: 已修复

```python
track = raw.get("track_score", 0)      # key 存在但值为 null → 返回 None
density = raw.get("density_score", 0)  # 同上
# ...
item["ai_score"] = track + density + insight + timeliness  # None + int → TypeError
```

`dict.get(key, default)` 只在 key **不存在**时返回 default。如果 LLM 返回 `{"track_score": null}` 且 Pydantic 校验耗尽重试次数，raw dict 带着 `None` 值到达这里 → `None + int` 抛 `TypeError`。

**修复**: `raw.get("track_score") or 0` — 同时覆盖 key 不存在和 key 存在但值为 null。

### ✅ BUG #A3: CSV 文件名用 `datetime.now()` 而非传入的 date 参数

**文件**: `tools/scrapers/base.py:220` | **严重度**: 中 | **状态**: 已修复

`_write_csv()` 无条件调用 `datetime.now()` 生成文件名。用户调用 `scraper.run('2026-06-23')` 回填历史数据时，CSV 会写成今天的文件名 → 不同日期的文件互相覆盖，文件名与内容不一致。

**修复**: `_write_csv` 新增 `date` 参数（默认空，fallback 到 `datetime.now()`），`run()` 传入 scraper date。

### ✅ BUG #A4: float zero `0.0` 绕过 Bundle ID fallback

**文件**: `tools/scrapers/base.py:180` | **严重度**: 中 | **状态**: 已修复

```python
bundle_id = mapped.get("bundle_id", "")
if bundle_id == "0" or not bundle_id:
    bundle_id = f"fallback:{game_name}"
```

API 返回 `{"bundle_id": 0.0}`（JSON 数字）→ `str(0.0) = "0.0"` → `"0.0" != "0"` 且 `"0.0"` 为真 → fallback 被绕过。

**修复**: `bundle_id = mapped.get("bundle_id", "").strip()` + `if bundle_id in ("0", "0.0", "") or not bundle_id:`

## 📊 根因分析

三个高频模式：

1. **Python 边界条件** (#A2, #A4): `dict.get(key, 0)` 对 `None` 值的语义误解；`str(0.0)` 的结果与直觉不符

3. **复制粘贴散度** (#A1, #A7): 同一逻辑在多处重复实现，细节逐渐分叉
